HTML Tags I've Learned

<!DOCTYPE>
Defines what type of document the document is. For our purposes, usually an HTML document (for now).
<html></html>
Contains everything we want in the document (i.e., all of the content must be in between the opening and closing tags).
<head></head>
Contains information about the document and links to our stylesheets, which will be CSS. This is all content hidden to the user, unless they inspect the page code.
<body></body>
Contains all of the content on the webpage that we want to display to the user.
<!-- -->
Also known as the comments tag, this allows us to write comments within the code in order to unofficially label or communicate the purpose of certain sections of code without affecting visible content.
